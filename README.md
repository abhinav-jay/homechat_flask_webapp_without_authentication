# homechat_flask_webapp_without_authentication
You can use this if you want to run a chat application or a forum on your computer and don't want registration or login.

## Installation
```bash
git clone https://github.com/abhinav-jay/homechat_flask_webapp_without_authentication
cd homechat_flask_webapp_without_authentication
python app.py
```
You will then have to stop than using Ctrl+C and then when you want to run the chat application you can use
## Usage
```bash
flask --app app run --host=0.0.0.0
```
If you want to use this with authentication, go [here](https://github.com/abhinav-jay/homechat-flask-webapp)
